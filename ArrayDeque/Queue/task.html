You are given a list of integers. Your task is to implement a queue using ArrayDeque in Java and then perform the following operations:

  1. Enqueue: Add an element to the end of the queue.
  2. Dequeue: Remove an element from the front of the queue.
  3. Peek: Examine the first element of the queue without removing it.

Write three methods—enqueue, dequeue, and peek—to perform these operations.

Input:

The enqueue method takes an integer num as input, which is the number to be added to the end of the queue.

The dequeue and peek methods do not take any input.

Output:

The enqueue method does not return anything.

The dequeue method should return the number that was removed from the front of the queue.

The peek method should return the number at the front of the queue without removing it.


Sample Input:
3
4
5
6

Sample Output:
4
4
5
5
6
6



Memory limit: 256 MB
Time limit: 5 seconds
